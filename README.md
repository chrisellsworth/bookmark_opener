# bookmark_opener

Chrome plugin to open the N-th bookmark in your toolbar using a keyboard shortcut. (Analogous to Cmd-Opt-{1,2,3,…} in Safari.)

## Installation

1. Navigate to [chrome://extensions/](chrome://extensions/)
1. Enable _Developer mode_
1. Click _Load unpacked_
1. Choose `boomark_opener` folder

## Usage

- Due to Chrome restrictions, only Opt-{1…4} (or Alt-{1…4}) will be bound by default.
- Navigate to [chrome://extensions/shortcuts](chrome://extensions/shortcuts) to add shortcuts for additional indices.
